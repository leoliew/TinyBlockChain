## 参考视频

* https://www.savjee.be/2017/07/Writing-tiny-blockchain-in-JavaScript/

## Feature

* 简单的共识机制
* 校验区块链完整性 (防止篡改)
* 创建区块链钱包 (非对称加密算法)
* 签署交易
